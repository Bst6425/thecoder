Introduction to HTML

As you know till now,Html is a hypertext markup language.It is used to create structure of webpages or websites.html is a not a programming language,it is a markup language.


PROGRAMMING LANGUAGE

Programming language is a language used to communicate with computer,it is used to give command to computer to perform any task .It is need to compile.Examples are java,c++.

SCRIPTING LANGUAGE

scripting language is used to connect two languages.It is a part of programming language but need not to complie.Examples are javascript.

MARKUP LANGUAGE

Markup language is just used to display, structure and format data.It is not give command to computer and need not compile.Examples are html.

COMPILATION

Compilation is a process of converting our written code in any language into binary language that is 0 and 1 type language that is what computer understand as no current/switch off is 0 and flow of current/switch on is 1.

Html syntax

<!docktype html>
<html>
<head>
<title>title of webpage </title>
<body>
<h1>heading 1 </h1>
<p> paragraph </p>
</body>
</html>

# docktype tag tells browser about version of html here is 5.

# html tag tells browser that it is a html document.

# head tag contains information of webpage that does not display on webpage,such as title, internal css,etc.

# title tag is used to define title of webpage.

# body is main part,it is a part which display on webpage.It is main workarea.

# h1 tag is used to create heading and p tag is used to paragraph.

# closing body tag is used to end part of body and similarly closing html tag is used to end html document.
